# MinLisp
[![Build Status](https://travis-ci.org/namachan10777/minlisp.svg?branch=dev)](https://travis-ci.org/namachan10777/minlisp)

minimum lisp implementation in C.
This program depends only C standard library.
# Language Function
 * Floating number
 * Function defination
 * Variable
 * Lexical scope
 * cons, cdr, car
 * if
 * quote
